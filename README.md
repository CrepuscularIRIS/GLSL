# BSL Enhanced

A visual enhancement shader pack based on **BSL enhanced for 1.7.10**.

## Features

- **Nether**: Ported fog and noise textures from **Complementary**
- **End**: Ported black hole and accretion disk effects from **Solas**
- **Water Surface**: Photon-style water waves and reflections
- **Aurora**: Ported aurora effects from **Lux**
- **Twilight Forest (World7)**: Uses Lux water configuration with aurora enabled
